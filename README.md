# 🎮 Player Manager Bot

🤖 بوت تلغرام متكامل للتحكم بسيرفرات ماينكرافت بيدروك (Bedrock) مع دعم السيرفرات المتعددة والذكاء الاصطناعي.

---

## ✨ المميزات

| الميزة | الشرح |
|--------|-------|
| 🖥️ **سيرفرات متعددة** | إدارة أكثر من سيرفر في نفس البوت |
| ➕ **إضافة سيرفر** | إضافة سيرفر جديد بسهولة عبر البوت |
| 📁 **قائمة سيرفراتي** | عرض وإدارة جميع السيرفرات المضاف |
| ⚙️ **إعدادات متقدمة** | كل سيرفر له إعداداته المستقلة |
| 🧠 **الذكاء الاصطناعي** | دعم ChatGPT في تلغرام وداخل السيرفر |
| 🔒 **نظام السجن** | سجون متعددة مع اختيار عشوائي |
| ⚠️ **نظام الإنذارات** | إنذارات تلقائية مع حظر بعد 3 إنذارات |
| 🚶 **حركة تلقائية** | البوت يتحرك عشوائياً داخل السيرفر |
| 📿 **أذكار تلقائية** | إرسال أذكار في شات السيرفر |
| 💬 **رسائل محاكاة** | إرسال رسائل عشوائية في الشات |

## 📋 الأوامر

### 🟢 أوامر عامة (للجميع)

| الأمر | الوظيفة |
|-------|---------|
| `/start` | عرض القائمة الرئيسية |
| `/help` | عرض المساعدة |
| `/status` | عرض حالة السيرفر |
| `/connect` | توصيل البوت بالسيرفر |
| `/disconnect` | فصل البوت من السيرفر |

### 🔴 أوامر السيرفر (للاعبين OP فقط)

| الأمر | الوظيفة |
|-------|---------|
| `/امر <أمر>` | تنفيذ أمر ماينكرافت |
| `/اعطي <لاعب> <شيء> <كمية>` | إعطاء أدوات |
| `/سجن <لاعب>` | سجن لاعب |
| `/سجن حدد` | تحديد موقع السجن |
| `/سجن مسح` | مسح جميع السجون |
| `/سجن قائمة` | عرض السجون |
| `/فك سجن <لاعب>` | فك السجن |
| `/كتم <لاعب>` | كتم لاعب |
| `/فك كتم <لاعب>` | فك الكتم |
| `/طرد <لاعب> <سبب>` | طرد لاعب |
| `/حظر <لاعب> <دقائق>` | حظر لاعب |
| `/انذار <لاعب>` | إعطاء إنذار |
| `/نظف` | حذف الأدوات الملقاة |

## 🛠️ كيفية الاستخدام

### 1️⃣ إضافة سيرفر جديد

1. اضغط على **➕ إضافة سيرفر**
2. أرسل عنوان السيرفر (IP)
3. أرسل المنفذ (Port)
4. يتم إضافة السيرفر تلقائياً باسم `السيرفر 1`, `السيرفر 2`...

### 2️⃣ اختيار سيرفر

1. اضغط على **📁 سيرفراتي**
2. اختر السيرفر المطلوب
3. تظهر لك لوحة الإعدادات

### 3️⃣ إعدادات السيرفر

| الإعداد | الوظيفة |
|---------|---------|
| ✏️ تعديل اسم البوت | تغيير اسم البوت داخل اللعبة |
| ✏️ تعديل اسم النداء | تغيير الكلمة المنادى بها |
| 🧠 الذكاء في السيرفر | تفعيل/تعطيل الذكاء داخل السيرفر |
| 📝 إعدادات الرسائل | تعديل رسائل السجن، الإنذار... |
| 🚶 إعدادات الحركة | تفعيل/تعطيل الحركة، نصف القطر، السرعة |
| ⏰ إعدادات الوقت | وقت الأذكار، وقت المحاكاة |
| 💬 رسائل تلقائية | تعديل رسائل المحاكاة والأذكار |
| 🔌 توصيل البوت | تشغيل البوت على السيرفر المختار |
| 🔴 فصل البوت | إيقاف البوت |
| 📊 حالة السيرفر | عرض اللاعبين المتصلين |
| ❓ المساعدة | عرض الأوامر |
